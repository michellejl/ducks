# Ducks

### Install dependencies:
```sh
yarn
```

### Build development environment:

```sh
gatsby develop
```

### Compile styles (sass)
```sh
sass --watch src/components/sass/layout.scss src/components/layout.css
```

### Build production version:
```sh
gatsby build
```
